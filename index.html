Projects

CD33 SNP rs12459419 & Alzheimer’s Risk Study

A multi-omics bioinformatics study investigating how the rs12459419 SNP in the CD33 gene alters gene regulation in microglia. By integrating GTEx, ENCODE, and 1000 Genomes data, the study links the T allele to reduced CD33 expression and increased exon 2 skipping — with implications for Alzheimer’s disease risk and therapeutic targeting.

Gene Data Integration Tool

A Perl-based script that fetches gene data from Ensembl, NCBI, and UCSC, storing results in a SQLite database and generating a comprehensive PDF report. Supports exon structures, genomic coordinates, and gene descriptions.

Codon Usage Frequency Tool

Web-based app that processes DNA sequences, translates them to protein, and reports codon usage frequency. Built with Python CGI, HTML, JavaScript, and jQuery for interactive analysis.

LCS Algorithm Analyzer

A Python tool that compares brute-force and dynamic programming approaches for solving the Longest Common Subsequence (LCS) problem. It reads input strings from a file, performs pairwise comparisons, and outputs LCS lengths, sequences, and execution time. Useful for benchmarking algorithm performance and complexity.
