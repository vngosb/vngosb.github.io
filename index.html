Personal Projects

RNA Seq Differential Expression Pipeline

Built a reproducible RNA-seq pipeline (SRA raw data, data processing, QC, trimming, Salmon, DESeq2) to identify differentially expressed genes between tumor and normal samples. Produced PCA, heatmap, and volcano plots to highlight expression differences.

School Projects

CD33 SNP rs12459419 & Alzheimer’s Risk Study

Multi-omics bioinformatics study investigating how the rs12459419 SNP in CD33 alters gene regulation in microglia. Integrates GTEx, ENCODE, and 1000 Genomes data to link the T allele with reduced CD33 expression and increased exon 2 skipping, with implications for Alzheimer’s disease risk.

Gene Data Integration Tool

Perl-based tool fetching gene data from Ensembl, NCBI, and UCSC, storing results in SQLite, and generating comprehensive PDF reports including exon structures, genomic coordinates, and gene descriptions.

Codon Usage Frequency Tool

Web-based app that processes DNA sequences, translates to protein, and reports codon usage frequency. Built with Python CGI, HTML, JavaScript, and jQuery for interactive analysis.

LCS Algorithm Analyzer

Python tool comparing brute-force and dynamic programming approaches for Longest Common Subsequence (LCS) problem. Reads input strings, performs pairwise comparisons, and outputs LCS lengths, sequences, and execution time for performance benchmarking.

Strassen vs Standard Matrix Multiplication Analyzer

Python script comparing standard matrix multiplication with Strassen’s algorithm. Reads input matrices from a text file, outputs result matrices, computation time, and multiplication count. Highlights trade-offs between brute-force and divide-and-conquer approaches in linear algebra.
